## 📌 About the Project

**Smoothify** is a front-end web project that simulates a smoothie shop website. It features a fun animated homepage, a product menu with smoothie cards, and an About & Contact page — all built without any frameworks, using only HTML and CSS.

---

## ✨ Features

- 🎬 **Animated Homepage** — fruits fly in and blend into a smoothie on page load
- 🧃 **Menu Page** — smoothie cards with images, descriptions, and prices (PKR)
- 📄 **About & Contact Page** — shop info and contact details
- 📱 **Responsive Design** — works on mobile and desktop
- 🎨 **Custom CSS Styling** — gradient backgrounds, hover effects, smooth transitions
